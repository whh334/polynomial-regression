# 多项式回归项目

这是一个使用 scikit-learn 实现多项式回归的示例项目，用于演示机器学习中的多项式回归算法。

## 📋 项目描述

本项目使用多项式回归拟合非线性数据，展示了如何：
- 生成示例数据
- 使用 PolynomialFeatures 进行特征转换
- 应用线性回归进行多项式拟合
- 可视化拟合结果

## 🛠️ 技术栈

- Python 3.11
- NumPy：数据处理
- Matplotlib：数据可视化
- scikit-learn：机器学习库
